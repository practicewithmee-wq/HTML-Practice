## **Color And Background systems**
- Foreground = color: colorname;

- background = background-color: colorname;
- RGB (RED, GREEN, BLUE) = color: rgb(0,0,0) Range = (0-255)
- Hex   = #ff00ff; Range = (a-f)
- Alpha Channel (transparent) = rgba(255,0,0,0.2)  Range = (0.0-1.0)
